## Contributing

### Installing and building SDKs

```
yarn 
yarn wrun build
```

### Generating SDK's

```
yarn generate
```

### Updating OpenAPI spec

All specifications are kept in `.openapi` folder:

https://github.com/redhat-developer/app-services-sdk-js/tree/main/.openapi
